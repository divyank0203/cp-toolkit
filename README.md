# Competitive Programming Toolkit

A lightweight toolkit designed to help competitive programmers test, debug and validate their solutions locally before submitting to online judges.

The toolkit provides utilities for generating custom test cases, running programs against multiple inputs, and comparing expected vs actual outputs.

## Features

- Custom test case generator
- Batch execution of solutions against multiple test files
- Output comparison for debugging wrong answers
- Library of commonly used algorithm templates
- Designed for fast local testing during contest practice

## Motivation

While practicing competitive programming, debugging incorrect solutions can be time-consuming. This toolkit aims to simplify the workflow by allowing programmers to quickly generate tests, run solutions, and validate correctness locally.

## Tech Stack

- Node.js
- JavaScript
- C++ algorithm templates
